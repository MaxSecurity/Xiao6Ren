首先运行 安装模块
```
pip3 install -r requirements.txt
```
运行程序 
```
python3 main.py 
```
 
运行成功会出现 阳历时间 阴历时间，并且会让你添加月 日 时辰 数字如果你是报数起卦月日输入处可以直接使用随机数字即可
```angular2html

辛丑年丙申月庚戌日

子时 23:00-01:00：1 
丑时 01:00-03:00：2
```
