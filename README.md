### 小六壬 起卦程序

首先运行 安装模块
```
pip3 install -r requirements.txt
```
运行程序 
```
python3 main.py 
```
 
运行成功会出现 阳历时间 阴历时间，并且会让你填写月 日 时辰 数字
如果你是报数起卦月日输入处可以直接使用随机数字即可

三宫所属： 速喜 ☯ 赤口 ☯ 留连

【小六壬合宫取象】
大安大安，可取犯小人，可取双手，取两人。可取肝硬化。
大安流连，身体方面可以取肠道的问题，消化不良，流连取肠胃，大安取食物，大安克流连，是肠胃难消化这些食物。
大安速喜，可以取酸辣，可以取肝火旺，颈椎炎，肩周炎，因大安主直，速喜为炎症，大安又可以取经脉，神经，因木有升发之意，可取神经，那么速喜为炎症，也可以取神经炎， 为皮肤过敏，为痘痘。
大安赤口，可以取骨刺增生，大安可去颈椎骨，脊椎骨，手臂等，赤口可以取骨刺，这是骨刺增生的类象，同时也可以取打架斗殴，或者摔跤，伤到了经脉，也可以取呼吸道，大安直，取呼吸道，赤口是口，可取嘴巴或鼻孔。其他类象亦可以取医生拿刀、开刀。
大安小吉，可取风湿病，大安为巽，是风，小吉为水，是湿，故而取风湿病。同时也可以取酸，可以取肝腹水。
大安空亡，可取肝癌，可以取颈椎、手臂、等地方有浮肿，或手臂有肿瘤。可取家中的床上。（前面有说过，空亡为肿胀）
流连流连，取纠缠不休，争辩不停，便秘，消化不良。
流连速喜，为肠胃炎，为近视眼，因为速喜为眼睛，流连主虚虚实实，这是近视眼的一个类象。妇科病，大姨妈推迟，为没有精神，为想睡觉。
流连赤口，这是呼吸道困难，若是在配上小吉的话，都可以取喉咙里有痰。回到主题，流连赤口，流连为纠缠为黏住，赤口是嘴巴又是鼻孔，这也是呼吸道困难的类象，同时也可以代表胃痛，赤口主痛，所取胃痛。同时也是惊吓，虚病、邪病。剖腹产，流产，白带多， 子宫有异味。
流连小吉，可取胃酸少，消化不良，也可取肾虚，胆虚，胆小。流连空亡，为胃癌，为没食欲，为精神病，为妄想症。
速喜速喜，火气很大，脸上多有雀斑，痘痘等，心脏病。
速喜赤口，为肺炎，为急性病，为做手术，为检查，赤口为手术，速喜为眼睛，可取检查的意思，为呼吸不顺。
速喜小吉，为胆炎，为肾炎，为血液性疾病，为大姨妈。
速喜空亡，坟前烧香（因空亡为坟，所以无救），爱妄想，健忘症，想得多做得少，胃炎， 神经衰弱，心脏衰竭，瞎子，易学者，尼姑，算命，道士。
遇见两个一样的卦，犯小人居多，同时也主朋友，若是投资的话，多为合作人或者竞争者， 单宫类象以上我也有描述很多，这只是笔者想到的，其他的类象还有更多，在此就不一一举例。
